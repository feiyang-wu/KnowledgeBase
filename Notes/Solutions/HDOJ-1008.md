# Elevator
## 题目大意
表达式求值，纯水题
## 解题思路
`Sum = N * 5 + Upstairs * 6 + Downstairs * 4`
## 细节问题
无
## 代码实现
```
#include <iostream>
#include <string>
#include <map>
#include <cmath>
#include <vector>
#include <algorithm>

using namespace std;

int main()
{
	int n;
	while (scanf("%d", &n) && n != 0)
	{
		int sum = 5 * n;
		int pos = 0;
		for (int i = 0; i < n; i++)
		{
			int floor;
			scanf("%d", &floor);
			if (floor > pos)
			{
				sum += ((floor - pos) * 6);
				pos = floor;
			}
			else
			{
				sum += ((pos - floor) * 4);
				pos = floor;
			}
		}
		printf("%d\n", sum);
	}
	return 0;
}
```
