# Digital Roots
## 题目大意
计算数字的各个位的和，循环迭代直至和为个位数。

如：
```
36 -> 3+6=9                     output: 9
39 -> 3+9=12 -> 1+2=3           output: 3
1234 -> 1+2+3+4=10 -> 1+0=1     output: 1
```
## 解题思路
模拟，纯水题
## 细节问题
无
## 代码实现
```
#include <iostream>
#include <string>
#include <map>
#include <cmath>
#include <vector>
#include <algorithm>


using namespace std;

int main()
{
	string s;
	while (cin >> s && s != "0")
	{
		int sum = 0;
		for (int i = 0; i < (int)s.size(); i++)
		{
			sum += (s[i] - '0');
		}
		int ans = sum;
		while (ans >= 10)
		{
			ans = 0;
			while (sum >= 10)
			{
				ans += (sum % 10);
				sum /= 10;
			}
			ans += sum;
			sum = ans;
		}
		printf("%d\n", ans);
	}
	return 0;
}
```
